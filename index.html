HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta tag defines metadata about an HTML document. It is the information about the dtaa.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are 3 types of css design, inline, internal and external
  • The inline css apply to a specific HTML tag, it uses thestyle attribute rule to a particular element
  • The internal css also apply directly in the HTML file using the style element in the head sector
  • The external css keeps the stylesheet in a separate folder and then link to the HTML file using the link attribute

Git

  • git status: checks what branch we are currently
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: gets updates from the remote repo
  • git add: we use the command to include changes of a file into the next commit
  • git push: we use this to upload our commits to the remote repository
  • Git flow steps: add, commit, and push your work to branch. Then open pull request, merge your changes with the main branch and close issue

JavaScript

  • JavaScript is an object-oriented programming language.
  • A function is a set of instructions that tells the computer how to perfom a certain task.
  • A function does not execute automatically when the JavaScript file run but it executes when it is called.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
